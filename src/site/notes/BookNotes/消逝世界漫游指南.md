# Comments
通过亚当斯的文字，谁能不对鸮鹦鹉着迷呢？我也想知道，为什么这些巨大的鸟轻而易举地放弃了飞翔的冲动与渴望。好喜欢这本书，超过了对《[[BookNotes/银河系搭车客指南\|银河系搭车客指南]]》的喜爱，深深感受到了DNA的诙谐和可爱，又时刻被他流露出的真挚所感染。没想到他写非虚构也是这么行云流水，如果他能活到这个时代，会不会成为一位惊人的段子手或者vlogger？然而世界在走向消逝，这个世界逐渐失去了许多鲜活缤纷的物种，也失去了他。
