# Highlights

走在街上，我几乎听不懂擦肩而过的人们讲的话，也看不懂路过的招牌上写的单词。我就像一座坚固且移动的小岛穿过人群。有时，我会觉得自己的肉体就像某种监狱，仿佛一生经历过的所有记忆，和那些无法与记忆分离的母语一起被孤立、封印了起来。然而，孤立越是坚不可摧，意料之外的记忆就会越发鲜明，沉重得仿佛快要将我压倒。
