# Comments
有趣易懂的小册子，从时间的定义开始聊起，介绍了时间旅行的原理和面临的问题，以及“块状宇宙”所意味的未来的确定性与时间流逝的不存在。还有一点哲思：我看到的你是三万亿分之一秒之前的你，而时间真的存在于我们的头脑之外吗？

另外，许多年前就开始在不同书里读到玻尔兹曼自杀的故事，有机会想去他的墓碑看看。

祖先们的计时系统是基于天空的运行。前后两次日出之间的间隔构成一天，这个时间间隔再分成小时、分钟和秒（后两者原先叫“一级小部分”[first minute]和“二级小部分”[second minute]，所以今天我们把二级小部分称作second[秒]）。祖先们把七天当成一个单元，叫“星期”，并用天空中运行的七个天体来给每一天命名，如SATUR[N]day（星期六/土曜日，）、SUNday（星期天/日曜日）、MO[O]Nday（星期一/月曜日）等。

这意味着你绝不会看到现在的月亮，而只会看到它1.3秒前的样子——进入你眼睛的是1.3秒前离开月亮的光。日常生活中你和别人打交道也是一样：你距离某人1米远时，看到的不是现在的他，而是三万亿分之一秒前的他，那是光线从他脸上离开的时候。光要花时间穿越空间，这个情况就意味着根本就没有一致的“现在”：我们看到的永远是过去，绝不会是现在。

在与科学机构和自身的精神状况进行了多年的斗争后，玻尔兹曼于1906年夏天上吊自杀，当时他还在和家人一起度假。他没有留下遗言。人们把玻尔兹曼埋葬在了维也纳的一处公墓里，这里也安葬着贝多芬和勃拉姆斯。他的墓碑上刻着一条热力学方程，至今人们还在用它计算熵值。

那么，无序性若一直增加，宇宙会发生什么呢？最终，整个宇宙会达到一种无序程度最大的状态，不会再有能量来产生哪怕很短暂的一点秩序，也不会有恒星、行星或者生命。好消息是，宇宙要达到这种状态，所需的年数至少是1后面跟上100个零。天文学家把它称作宇宙的“热寂”（有时候还叫“大冻结”或“大冷寂”）。 
{ #ref-24369}


---
众所周知，牛顿认为引力是一种力。引力把苹果吸引到地上，也托住沉重的行星绕太阳运转。然而爱因斯坦却另有想法，他认为实际上根本没有发生吸引。相反，引力只是时空弯曲的结果。 
{ #ref-47393}


---
人们经常用一个比方来描述这个观点。我们把一张四角拉直紧绷的床单想象成时空结构，把一个保龄球放在它的中心当作太阳。这个保龄球会在床单上造成一个凹陷，如果再放上一个网球，你就可以让这个网球沿凹陷的边缘滚动，这样它就绕着保龄球旋转了，就像地球绕太阳旋转那样。关键在于这两个球之间不存在牛顿式的吸引力。小球没有受大球的牵引，它只是沿着大球在床单上造成的曲线路径运动。这就是爱因斯坦对于引力的真正机制的直觉。 
{ #ref-41132}


---
你也是一个时间旅行者。时间膨胀意味着，每次你加快速度，都会比你保持静止时更快到达未来。 
{ #ref-63932}


---
要理解这种引力时间膨胀，还得回到第5章中那个床单和保龄球的比方上。床单代表“时空”这种遍及整个宇宙的结构。把保龄球放在床单中央代表大质量天体，如太阳，床单的中央就会下陷。物理学家把这种凹陷叫“引力井”。 
{ #ref-4464}


---
一般而言，你的脚比你的脸要更接近地面，这意味着脚更深入地球的引力井，于是，时间在脚踝处就比在眼睛处走得慢。你的脚趾就是时间穿越者。如果一个人的寿命一般为80岁，那么他的脚趾就会比舌头早约百万分之0.5秒到达未来。我们的头也会以不同的速度穿越到未来，取决于各人的身高。其他条件都一样时，身高1.5米的人的脑子会比身高1.8米的人的脑子更快到达未来。 
{ #ref-285}


---
描绘虫洞最简单的办法是：用一张普通的纸来表示时空，地球在这张纸的一端，另一端则是半人马座比邻星，它是除太阳之外离我们最近的恒星。要在这两个地方穿行，你一般要跨越整张纸，这是一场40万亿千米或说4.2光年的挠头旅程。但是，我们在前几章已经看到，时空可以被操纵、被弯曲。如果把这张纸对折，你的目的地一下子就变得很近了。纸的两端之间的通道能让你立即到达那边，这通道就是虫洞。 
{ #ref-12655}


---
看待这个概念的另一种方式是把块状宇宙想象成一本书。如果你从外面就能看这本书，你会看到过去和未来皆已发生。结局早已写定。这本书本身是静止不变的，就像个混凝土砌块似的放在那里（这也是“块状宇宙”一名的由来）。但是，我们却困在书的里面。我们在一页一页翻动这些静态的书页时，会把每张新页面（每个新时刻）和上一页做比较，也必然会感受到变化，于是这有了这样一个印象：我们被时间的河流带着前进，这条河不停地从过去经现在流向未来。但我们不会说空间过去了或流逝了。时间其实也不会。时间的流逝是我们想象的虚构。

---
我们每个人会通过不同的路径穿过这本时空之书，取决于我们走得多快。如果我比你更快地穿过时空，那么我将以更快的速度浏览完这些页面，更快地到达“未来”这个末尾（时间膨胀）。我相对你的穿越速度越快，你我之间的差别就越大。你也可以把一页书窝成一个圈，让它和上一页相交（回到过去的时间旅行）。但你无法改变已经写在书页上的东西。这就是为什么在希特勒或你的祖父还是孩子的时候，你无法杀死他们。

----
在一个更哲学化的层面上，块状宇宙理论认为未来已经在那里等着你了。你无法改变你的经历，就像你在阅读此页时也无法改变本书的结尾。如果明天不是由今日发生之事创造的，那么人有自由意志的观点就也像时间流逝一样纯属幻景了。

----  
欢迎进入“现实论”，这种思想认为只有现实发生的才是可能发生的。可能的未来只有一个，它已然预先决定了。

-----
如果惠勒—德威特方程是正确的——这是一个大号的“如果”——那么空间和时间可能是涌现出来的结果，我们只在较大的尺度下能感觉到，但它们实际上在更深的层次上根本就不存在。如果一个观察者能像神一样从外部观察到宇宙，那他会看到一个静止不变、没有时间的宇宙，时间就只是一种我们这些困在宇宙内部的人才有的幻觉。有可能时间只存在于我们的头脑之中。而讽刺的是，这只有时间能告诉我们答案。
