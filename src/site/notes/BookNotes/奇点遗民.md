# Comments
> 完全忘了我早就读过《爱的算法》，重读几个短篇，还是再次被打动了。能看得出来刘宇昆比较偏爱的几个主题：爱，殖民与移民，劳工，东方文化。其中的几篇很像是黑镜里的故事，也像是不远的未来。最近一段时间看的科幻，都是天马行空浮光掠影，这本书给轻飘飘的幻想与妄想配上了一条与情感相连接的血管。
