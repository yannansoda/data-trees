# Comments
很喜欢路易塞利的文笔，哪怕看得出有刻意的技法，依然很喜欢。起始都是威尼斯的迷路，中间是在墨西哥城的行走，对葡萄牙语saudade翻来覆去的把玩，和《[[BookNotes/失踪孩子档案\|失踪孩子档案]]》一样掉了很多书袋，有点太多但还没到让人厌烦的程度。很久没读这种散文集了，像报了个散漫的知识分子观光团、但边走边迷茫、但又有点享受的感觉。
